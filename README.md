"# express-mvc-adminlte" 
For this project i combine :
- express V4.17.1
- AdminLTE 3.1.0
- MVC Structure

How to use this starter pack
- open terminal / command prompt(windows)
- npm install
- create database (mysql) & import db (crud.sql)
- change database setting in environtment file (.env)
- npm start
- done
